# GeoGebra 9 牟合方盖

## 涉及指令

- intersect 相交路径
- Arc 弧线
- point 描点
- rotate 旋转
- polyline 折线

```py
序列(折线(序列(旋转(描点(d, t), 90° a, z轴), a, 0, 4, 1)), t, 0, 1, 0.01)
```
