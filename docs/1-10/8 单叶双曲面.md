# GeoGebra 8 单叶双曲面

## 涉及指令

- mod 取余
- point 描点
- segment 线段

```py
序列(线段(描点(c, 取余(a, 1)), 描点(d, 取余(a + 0.2, 1))), a, 0, b, 0.01)
圆周((0, 0, 3), 3, z轴)
圆周((0, 0), 3, z轴)
```

或者

```py
曲面(线段((1, -1), (1, 1, 4)),360°,z轴)
```
