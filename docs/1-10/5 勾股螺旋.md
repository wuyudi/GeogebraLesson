title: 5 勾股螺旋
---

本次介绍了 2 种制作勾股螺旋的方法。涉及新指令 x() ,y(), last 最后元素，length 长度

指令如下

```py
迭代列表((x(a) + (x(a)² + y(a)²)^(-1 / 2) y(a) (-1), y(a) + (x(a)² + y(a)²)^(-1 / 2) x(a)), a, {(1, 0)}, b)

迭代列表(p+单位法向量(p-A),p,{B},20)

映射(多边形(a, (0, 0), c), a, l1, c, 最后元素(l1, 长度(l1) - 1))
```
